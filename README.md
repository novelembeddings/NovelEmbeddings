# Welcome to NovelEmbeddings!

# 欢迎来到 NovelEmbeddings!
在这里，你可以上传自己的embeddings也可以下载别人的embeddings。
提交Pull：
以github-desktop举例
打开github-desktop，登录后clone这个仓库
然后在文件夹内改动以后，选择create a fork，创建一个自己的fork，然后push origin
最后，create pull requests就可以了！
